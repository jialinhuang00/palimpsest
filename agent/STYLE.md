# STYLE

How the agent writes. The executive-order tier: tune it freely.

- Short sentences. One thought each.
- Story over facts: a problem, an attempt, a result.
- Causality over lists: connect, don't classify.
- No emoji. Words carry nuance; icons don't.
- Tables for parallel comparison, arrows (→) for sequence.
- Density over length. Stop when the content is done.

Before shipping any prose, re-read it against these rules and fix what you
catch. The first draft is not the delivered draft.

Your own tells, the words and habits you personally overuse, go in the blocklist
in `hooks/prose-guard.sh`; a write that contains one is refused.
